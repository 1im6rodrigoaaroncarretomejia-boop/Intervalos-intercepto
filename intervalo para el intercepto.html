Intervalo de confianza para β₀

Intercepto en regresión lineal simple — medición de línea base

Fórmula general para el intercepto
β̂0  ±  t(α/2, n−2) · Se · √( 1/n + x̄² / Sxx )

Toca cada término para entender su papel en el punto de partida:

β̂0 — intercepto estimado
t(α/2, n−2) — valor crítico
Se — error estándar
√(1/n + x̄²/Sxx) — factor de distancia
Selecciona un término para ver su significado y por qué afecta la certidumbre de nuestra línea base.
Ejercicio resuelto paso a paso
Se realiza un diagnóstico de estrés térmico en un procesador. Se mide la carga de trabajo de la CPU en porcentaje (X) y su temperatura resultante en °C (Y). Con n = 5 observaciones, queremos estimar el intervalo de confianza al 95% para el intercepto (β₀), que representará la temperatura base del equipo en reposo (0% de carga).

1
Calcular estadísticos base
Medias (x̄, ȳ) y Sumas de Cuadrados (Sxx, Sxy)
x̄ = 30, ȳ = 50.4
Sxx = 1000, Sxy = 780
2
Estimar los parámetros (Mínimos Cuadrados)
β̂₁ = Sxy / Sxx  |  β̂₀ = ȳ − β̂₁·x̄
β̂₁ = 780 / 1000 = 0.78
β̂₀ = 50.4 − (0.78)(30) = 27.0
Temperatura inicial estimada: 27.0 °C
3
Calcular el error estándar residual Se
SSE = Syy − β̂₁·Sxy  |  Se = √(SSE / (n−2))
Syy = 609.2
SSE = 609.2 − (0.78)(780) = 0.8
Se = √(0.8 / 3) ≈ 0.516
4
Calcular el Error Estándar del Intercepto SE(β̂₀)
SE(β̂₀) = Se · √( 1/n + x̄² / Sxx )
SE(β̂₀) = 0.516 · √(1/5 + 900 / 1000) = 0.516 · √(1.1) ≈ 0.541
5
Valor crítico y Margen de Error (E)
gl = n - 2  |  E = t(α/2, n-2) · SE(β̂₀)
t(0.025, 3) = 3.182
E = 3.182 · 0.541 ≈ 1.72
6
Construir el intervalo
IC = β̂₀ ± E
IC = 27.0 ± 1.72
Límite inferior: 25.28   Límite superior: 28.72
Conclusión del Diagnóstico
IC₉₅%(β₀) = (25.28 °C , 28.72 °C)
Con un 95% de confianza, la temperatura base del procesador en estado de reposo absoluto (0% de carga) se encuentra entre 25.28 °C y 28.72 °C. Esta es una línea base térmica saludable y certifica que el sistema de enfriamiento está bien calibrado desde el inicio.
